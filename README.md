# Leakage Detection in Water Networks using Gradient Boosting

## Business Overview
Water scarcity and resource waste are critical global challenges. This project provides a data-driven solution for detecting leakages in distribution networks, enabling proactive maintenance and reducing non-revenue water (NRW) losses.

## Key Technical Features

Algorithms: Evaluated XGBoost, Random Forest, and SVM for high-precision anomaly detection.

Causal Inference: Implemented SHAP values to identify the primary features (e.g., pressure drops vs. flow rate) driving leakage predictions.

Scalability: Built a modular pipeline in Python designed to handle real-time sensor data streams.

## Results

Achieved an F1-Score of 94%, prioritizing the reduction of False Negatives (missed leaks).

Distilled complex sensor data into actionable maintenance alerts.
